### Что за проект
Премиум-лендинг продажи диагностики бизнеса (медицинская метафора: бизнес =
пациент, маркетинг = диагностика). Палитра (тьма + зелёный `--vital` + золото) и
каркас блоков НЕ меняются. Стиль — строгий премиум (чёткие границы, малые/прямые
углы, угловые скобки, плотный болд, угловые скобки-рамки, иконки в капсулах,
дозированные эмодзи на декоре). Подробности — `DESIGN.md`.

### Стек и структура

акцент, дозированно), JetBrains Mono (тех-лейблы). Самохостинг через Fontsource.
- `src/styles/tokens.css` — токены (палитра), `src/styles/base.css` — сброс/типографика/
  `.btn`/`.ic-cap`/reveal/keyframes.
- `src/layouts/Base.astro` — layout + фон + reveal.
- `src/components/fx/Background.astro` — фон (свечение + квадраты-контуры).
- `src/components/Topbar.astro`, `src/components/blocks/*.astro` — блоки.
- `src/pages/index.astro` — собирает блоки; `src/pages/zayavka.astro` — анкета (пока заглушка).
- `src/assets/img/` — выгруженные картинки + `_manifest.json` (привязка к блокам).

### Картинки
Все 41 base64-картинки уже вынуты скриптом `scripts/extract-images.mjs` в
`src/assets/img/` (имена `img-NN-<hash>.jpg`). Привязка — в `src/assets/img/_manifest.json`
(поля `order`, `nearClass`, `alt`). Отдавать через `astro:assets`:
`import { Image } from 'astro:assets'; import x from '../../assets/img/img-NN-....jpg';`

### Гочи (важно)
- **Скриншот-тул (`preview_screenshot`) часто виснет** в этой среде, особенно при
  непрерывных CSS-анимациях и эмодзи. Не залипай на нём. Проверяй вёрстку и текст
  через `preview_eval` (DOM/геометрия/computed styles). Если нужен кадр — заморозь
  анимации инъекцией CSS `*{animation:none!important}` + `.rv{opacity:1!important}`,
  сделай `preview_resize`, и только потом скриншот (срабатывает не всегда).
- **Не выдумывай цифры/факты** (никаких «+38% конверсия», «health score 87»). На
  реальном сайте только то, что есть в исходнике.
- **Эмодзи** — только на новых декоративных UI-элементах, НЕ внутри дословного
  текста (иначе нарушишь правило неизменности).
- Не добавляй eyebrow/подзаголовки, которых нет в оригинале блока.
- Дев-сервер: `preview_start` по `.claude/launch.json` (имя `astro-dev`, порт 4321).
